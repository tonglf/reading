# 从缺陷中学习C/C++

《从缺陷中学习C/C++》这本书收集并整理了102个实例，这些实例都来自于工程一线实践，虽然大多数看起来像是初学者犯的低级错误，但实质上有一定的代表性，有的错误根源是对C++机制的不理解或者编译过程中的副作用，或者是对C++标准库的实现依赖。通过阅读这些实例，你可以对C++有更细致的理解。
